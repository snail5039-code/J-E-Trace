# J-E-Trace

J-E-Trace는 학생과 교사를 위한 **AI 채팅 기반 학습 기록 및 과제 관리 플랫폼**입니다.  
과제 수행 과정에서 발생하는 **학생의 AI 질문/응답 로그**, **과제 제출 내역**, **교사 평가**, **유사도 분석 결과**를 하나의 서비스 안에서 관리할 수 있도록 설계했습니다.

배포 주소  
https://j-e-trace-2r2vdkihn-snail5039-codes-projects.vercel.app/

---

## 프로젝트 소개

기존 과제 제출 시스템은 최종 결과물만 확인하는 경우가 많아,  
학생이 어떤 방식으로 학습했고 어떤 질문을 하며 과제를 수행했는지 파악하기 어렵습니다.

J-E-Trace는 이 문제를 해결하기 위해 다음 흐름을 지원합니다.

- 학생은 과제를 확인하고, 필요 시 AI와 대화하며 과제를 수행합니다.
- AI 사용 내역은 질문/응답 로그로 저장됩니다.
- 학생은 최종 답안을 제출합니다.
- 교사는 제출물, AI 로그, 유사도 분석 결과를 함께 확인하고 평가합니다.
- 관리자는 교사 계정 승인 및 운영 관리를 담당합니다.

즉, 단순 제출 플랫폼이 아니라  
**학습 과정 추적(Trace)** 과 **평가 보조**까지 포함한 교육 지원 서비스입니다.

## 주요 기능

### 1. 학생 기능
- 회원가입 및 로그인
- 본인 반에 해당하는 과제 목록 조회
- 과제 상세 확인
- 과제 내용 작성 및 최종 제출
- AI 허용 과제에 대해 AI 질문/응답 진행
- 제출 완료 후 제출 상태 및 점수 확인
- 최근 학습 로그 확인

### 2. 교사 기능
- 교사 회원가입 및 로그인
- 담당 반 기준 학생/과제 관리
- 과제 생성 및 상세 관리
- 학생 제출물 조회
- AI 대화 로그 확인
- 학생 간 / AI 로그 기반 유사도 분석 결과 확인
- 점수 및 교사 코멘트 입력
- 본인 프로필 및 담당 정보 관리

### 3. 관리자 기능
- 관리자 로그인
- 교사 회원가입 승인
- 서비스 운영용 계정 관리

---

## 서비스 대상

- **학생**: 과제를 수행하고 AI와 대화하며 학습 과정을 기록
- **교사**: 학생 제출물과 AI 사용 흐름을 함께 확인하며 평가
- **관리자**: 교사 계정 승인과 기본 운영 담당

---

## 핵심 포인트

- **AI 대화 기반 학습 기록**
  - 학생의 질문과 AI 응답을 기록하여 학습 흐름 확인 가능
- **과제 제출 + 평가 통합**
  - 제출 상태, 점수, 코멘트까지 한 흐름으로 관리
- **유사도 분석 지원**
  - 학생 간 제출물 비교 및 AI 로그 기반 분석 참고 가능
- **권한 분리**
  - 학생 / 교사 / 관리자 역할별 화면과 기능 분리
- **반(Class) 기준 관리**
  - 학생 과제 노출, 승인, 관리 범위를 반 기준으로 구분
